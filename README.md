# Himanshi Gupta | Elite Ayurvedic Physician Portfolio

> **Redefining Healthcare through the Fusion of Ancient Wisdom and Clinical Precision.**

Welcome to the digital identity of **Dr. Himanshi Gupta**, a BAMS graduate and healthcare innovator. This portfolio is a high-performance, emotionally intelligent digital experience designed to showcase her journey, expertise, and revolutionary approach to modern Ayurvedic medicine.

---

## 💎 Elite Design & Experience

This project isn't just a portfolio; it's a "Healing Aura" engine. Every interaction is designed to feel calm, organic, and professional.

### Key Features

- **🌀 Healing Aura Engine**: A custom cursor-following light effect that creates a soothing, reactive atmosphere using `framer-motion`.
- **🚀 Dynamic Elite Navbar**: A scroll-triggered navigation system that remains hidden at the Hero entry for maximum impact, sliding in with a premium glassmorphic effect as you explore.
- **🌊 Ultra-Smooth Motion**: Powered by `Lenis`, the site features luxurious, physics-based smooth scrolling for a modern feel.
- **🌗 Dual-Mode Aesthetic**: A meticulously crafted Dark and Light mode system that preserves high-class aesthetics and readability.
- **⚡ Performance Orchestrations**: A blend of `GSAP` and `Framer Motion` ensures that every transition is frame-perfect and fluid.

---

## 🛠 Tech Architecture

Built with a "Future-First" mentality using the most advanced web technologies available.

- **Framework**: [Next.js 16 (Turbopack)](https://nextjs.org/)
- **Core**: React 19
- **Aesthetics**: Tailwind CSS 4
- **Motion**: Framer Motion 12 & GSAP 3
- **Rendering**: Three.js & React Three Fiber (for future spatial diagnostics)
- **Typography**: Inter & Outfit (Google Fonts)
- **Navigation**: Lenis Smooth Scroll
- **Icons**: Lucide React

## 🚀 Getting Started

To run this elite experience locally:

1. **Clone the Repository**:
   ```bash
   git clone git@github.com:CodeSage4D/HimanshiDr.git
   cd HimanshiDr
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Launch the Engine**:
   ```bash
   npm run dev
   ```

4. **Experience locally**:
   Open `http://localhost:3000` in your browser.
